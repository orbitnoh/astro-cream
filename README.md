# Astro Cream

[astro.build](https://astro.build/)로 만든 SSG 블로그 템플릿입니다.  
단순히 마크다운을 렌더링하는 테마가 아니라, 글의 작성부터 관리까지 흐름을 지원합니다.  
마크다운을 날짜순 목록과 직접 큐레이션한 목차, 두 가지 보기 방식을 함께 지원합니다.

이 템플릿은 [Astro Cream CMS Tools](https://github.com/orbitnoh/astro-cream-cms-tools) VSCode 확장 프로그램과 함께 사용하는 것을 권장합니다.
